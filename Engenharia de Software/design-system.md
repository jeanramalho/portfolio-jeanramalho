# Attekita Dev - Design System

This document outlines the extracted design system tokens from [attekitadev.com](https://attekitadev.com).

## 🎨 Colors

### Backgrounds
- **Base Background**: `#07090E`
- **Surface**: `#111827`
- **Surface Alt**: `#1F2937`
- **Primary Brand**: `#DB2777` (Deep Pink)
- **White**: `#FFFFFF`
- **Black**: `#000000`

### Texts
- **Base Text**: `#FFFFFF` (White)
- **Primary Text**: `#DB2777` (Pink, mostly links)
- **Muted Text**: `#94A3B8`
- **Dim Text**: `#64748B`
- **Inverted Text**: `#000000` (Black)

## 🔤 Typography

**Primary Font Family:** `Montserrat`, sans-serif
**Fallback Font Family:** `Open Sans`, Helvetica, sans-serif

### Type Scale

| Element | Font Size | Line Height | Font Weight | Color |
|---------|-----------|-------------|-------------|-------|
| **H1** | 36px | 40px | 700 (Bold) | `#FFFFFF` |
| **H3** | 20px | 28px | 700 (Bold) | `#FFFFFF` |
| **Paragraph (p)**| 16px | 24px | 400 (Regular)| `#94A3B8` |
| **Links (a)** | 16px | 24px | 700 (Bold) | `#DB2777` |

## 📐 Layout & Spacing
- **Container Strategy**: Uses max-width layout (centered usually)
- **Border Box**: General reset `* { box-sizing: border-box; }` uses native defaults.

> Note: Hex color representations are based off extracted `rgb()` values.
